> **"이 사람이 우리 회사 외주 일감을 받아서 클라이언트 만족시킬 수준이 되나?"**

WEFLOW가 PT샵/필라테스/보험설계 같은 자영업자에게 사이트 만들어주는 1인 에이전시이므로, 평가자는 **"메뉴얼대로 빠짐없이 만드는 꼼꼼함"**을 본다.

## 핵심 평가 기준 (우선순위)

### 🥇 1순위 — 메뉴얼 지시 이행 능력 (50%)
- 메뉴얼 14페이지 = 평가 체크리스트
- "버튼 클릭 시 X로 이동" 같은 자잘한 지시가 50개+
- **창의력 X, 꼼꼼함 O**

### 🥈 2순위 — 데이터 흐름 동작 (25%)
- 사용자 폼 제출 → 관리자 페이지 실시간 반영
- 관리자 상태 변경 → 즉시 반영
- "버튼이 눌려야 함" 반복 강조 = 여기가 채점 포인트

### 🥉 3순위 — 모바일 반응형 (15%)
- 메뉴얼: "모바일 = PC 동일, 잘림/렉 없을 것"
- 한 줄로 적혀있지만 떨어뜨리는 기준

### 4순위 — 디자인/완성도 (10%)
- 기존 사이트보다 살짝 더 잘 만들기 (센스 점수)
- 마이크로 인터랙션, 모바일 깔끔함

   - 지난 시간 비활성화

3. **공통 컴포넌트화**

   - 하단바 1개로 모든 페이지 재사용
   - 평가자가 코드 보면 "구조 잡을 줄 안다" 판단
