# 데이터 모델 + API 설계

## Supabase 테이블

### `reservations` (예약)
```sql
create table reservations (
  id uuid primary key default gen_random_uuid(),
  name text not null,
  phone text not null,
  desired_date date not null,
  desired_time text not null,         -- '09:00', '09:30' 등
  custom_time text,                    -- 사용자 직접 입력 시간
  project_type text not null,          -- 랜딩페이지/홈페이지/랜딩&홈페이지/기타(케어플랜)
  industry text not null,              -- 업종 (PT샵, 필라테스 등)
  message text,                        -- 추가 요청사항
  status text not null default 'pending', -- pending/in_progress/done
  agree_terms boolean not null default false,
  created_at timestamptz default now()
);

create index reservations_status_idx on reservations(status);
create index reservations_created_idx on reservations(created_at desc);
```

### `inquiries` (문의 + 무료진단 공통)
```sql
create table inquiries (
  id uuid primary key default gen_random_uuid(),
  name text not null,
  phone text not null,
  project_type text not null,
  industry text not null,
  message text,
  source text not null,                -- 'diagnosis' | 'inquiry' | 'landing'
  status text not null default 'pending',
  agree_terms boolean not null default false,
  created_at timestamptz default now()
);

create index inquiries_status_idx on inquiries(status);
create index inquiries_source_idx on inquiries(source);
create index inquiries_created_idx on inquiries(created_at desc);
```

## API Route Handler

### POST `/api/reservations`
**Request**
```json
{
  "name": "유상훈",
  "phone": "010-1234-5678",
  "desiredDate": "2026-06-18",
  "desiredTime": "10:30",
  "customTime": null,
  "projectType": "홈페이지 제작",
  "industry": "PT샵",
  "message": "빠른 제작 부탁드립니다",
  "agreeTerms": true
}
```
**Response** `200`
```json
{ "ok": true, "id": "uuid..." }
```
**검증**:
- name/phone/desiredDate/desiredTime/projectType/industry 필수
- agreeTerms === true
- desiredDate가 오늘 이상

### POST `/api/inquiries`
- 위와 동일, `source: 'diagnosis' | 'inquiry' | 'landing'` 추가

### PATCH `/api/admin/reservations/[id]`
```json
{ "status": "in_progress" }
```
- 관리자 토큰 검증 (쿠키/세션)

### DELETE `/api/admin/reservations/[id]`
- 관리자 토큰 검증

### GET `/api/admin/export?type=reservations`
- 엑셀 파일 스트림 다운로드
- type: `reservations` | `inquiries`

### POST `/api/admin/login`
```json
{ "email": "badeagle85@gmail.com", "password": "1111" }
```
- 하드코딩 검증 → HttpOnly 쿠키 발급

## 폼 필드 — 메뉴얼 명세 그대로

### 공통 (예약/문의/진단 전부)
- 이름
- 연락처
- 제작종류 (셀렉트: 랜딩페이지 제작 / 홈페이지 제작 / 랜딩&홈페이지 제작 / 기타(WEFLOW케어플랜))
- 업종 (텍스트 입력)
- 추가요청사항 (textarea)
- ☑ 개인정보 수집 및 상담 동의 (체크박스)

### 예약 추가
- 달력 (날짜 선택)
- 시간슬롯 20개 (9:00~18:30, 30분 간격, 5×4 그리드)
  - **현재 시간 이전은 비활성** (오늘 날짜 선택 시)
- 원하시는 시간대 직접 입력

## 상태값

| 코드 | 한글 | 색상 |
|------|------|------|
| `pending` | 대기 | 회색 |
| `in_progress` | 진행중 | 파랑 |
| `done` | 완료 | 초록 |
| (삭제) | — | 빨강 (DELETE) |
