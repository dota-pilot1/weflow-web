# WEFLOW 과제 — 문서 인덱스

> 면접 과제 진행에 필요한 문서들을 정리한 폴더입니다.

## 폴더 구조

```
docs-for-프로젝트 개발 계획/
├── 개발 계획/
│   ├── TODO.md            ← 작업 진행 상황 (수시 갱신)
│   ├── 일정표.md          ← 4일 시간표
│   ├── 데이터모델.md      ← DB 스키마 + API 명세
│   └── 평가포인트.md      ← 메뉴얼 분석 + 차별화 전략
└── 참고 문서/
    ├── 채용공고.md         ← 회사 정보 + 포지션 상세
    ├── 원본메뉴얼정리.md   ← 메뉴얼 14페이지 페이지별 정리
    ├── 메뉴얼체크리스트.md ← 제출 전 점검용 체크박스
    └── 외부참고.md         ← 기존 사이트 URL + 외부 링크
```

## 사용법

### 작업 시작 전
1. `개발 계획/TODO.md` — 현재 진행 상황 확인
2. `개발 계획/일정표.md` — 오늘 할 일 확인

### 작업 중
- `참고 문서/원본메뉴얼정리.md` — 메뉴얼 원문 확인
- `참고 문서/외부참고.md` — 외부 URL/링크 확인
- `개발 계획/데이터모델.md` — DB/API 명세 확인

### 작업 후
- `개발 계획/TODO.md` — 진행 상황 갱신
- `참고 문서/메뉴얼체크리스트.md` — 체크리스트 점검

### 제출 직전
- `참고 문서/메뉴얼체크리스트.md` — **전체 체크박스 다 점검**
- `개발 계획/평가포인트.md` — 절대 빠뜨리지 말아야 할 것 확인
